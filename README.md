# 프로젝트명 : Caffeine the Last

# 팀명 : Dae-Choong

# 주제 : 카페인 중독에서 벗어나기

# 목적
늦게까지 공부와 일을 하면서 카페인이 함유된 음료를 자주 마시게 되고 카페인 중독에 빠져들게 된다.
이로 인해 수면 패턴이 엉망이 되고, 일상생활이 어려워진다.
그래서 카페인 중독을 치유하여 사용자 스스로가 카페인을 조절할 수 있게 만드는 것이 최종 목표이다.
